HBM's Nuclear Tech Mod - credits
 ================================

 This repository is a modern Forge 1.20.1 port of HBM's Nuclear Tech Mod.

 The original mod and its content were created by HbMinecraft and the original HBM project contributors. This repository is not the original 1.7.10 codebase; it is a separate restoration effort built against modern Forge.

 Thanks and attribution:

 - HbMinecraft, for the original mod this project is based on.
 - The Minecraft Forge team, for the modern modding API and toolchain this port is built on.
 - Mojang, for Minecraft itself and the official mapping set used by this workspace.
 - The broader Minecraft modding community, because a lot of old behavior is only understandable now thanks to years of forum posts, examples, bug reports, and archived code.
